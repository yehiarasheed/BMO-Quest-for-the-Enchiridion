# BMO Quest - Controls

## Movement Controls
- **W / Up Arrow**: Move forward
- **S / Down Arrow**: Move backward  
- **A / Left Arrow**: Strafe left
- **D / Right Arrow**: Strafe right
- **Shift (hold)**: Sprint (1.8x speed multiplier)
- **Spacebar**: Jump

## Camera Controls
- **Mouse Movement**: Look around (FPS-style)
- **Left Click**: Jump (alternative)
- **Right Click**: Toggle First Person ? Third Person view
- **Middle Mouse**: Toggle mouse look ON/OFF
- **C**: Toggle camera mode
- **ESC**: Toggle mouse look / Show cursor

## Debug Controls
- **U / O**: Rotate character left/right
- **P**: Print model positions and score
- **Z**: Wireframe mode
- **X**: Fill mode
- **Q**: Quit game

## Game Mechanics
- Collect cupcakes for points (10 pts each)
- Collect coins for points (5 pts each)
- Avoid the jelly - it will bounce you back!
- Score displayed in top-left corner (HUD)
